The statement

```text
The Schwarzschild radius is r_s = 2GM.
```

is incomplete unless the text explicitly states geometrized units with `c = 1`.

In SI or cgs units, the Schwarzschild radius is

```tex
r_s = \frac{2GM}{c^2}.
```

The expression `r_s = 2GM` is correct only in geometrized units where `c = 1`. If no such convention has been declared, the expression is dimensionally inconsistent.

A corrected statement is:

```text
In ordinary units, the Schwarzschild radius is r_s = 2GM/c^2. In geometrized units with c = 1, this becomes r_s = 2GM.
```
